# Configuration and Setup

After installation, the WARP4 Charger is configured via its web interface. Guidance for the
first steps (establishing a connection and opening the web interface) can be found under
[First Steps](/docs/tutorials/first_steps).

The following settings are available in the WARP4 Charger firmware:

 * Charger
   * [Settings](/docs/webinterface/wallbox/settings)
   * [Charging Status](/docs/webinterface/wallbox/charge_status)
   * [ISO 15118](/docs/webinterface/wallbox/iso15118)
   * [Charge Tracker](/docs/webinterface/wallbox/charge_tracker)
   * [Automation](/docs/webinterface/wallbox/automation)
 * Energy Management
   * [Energy Meters](/docs/webinterface/energy_management/energy_meters)

   * [Chargers](/docs/webinterface/energy_management/wallboxes)

   * [Dynamic Tariff](/docs/webinterface/energy_management/dynamic_tariffs)
   * [Solar Forecast](/docs/webinterface/energy_management/solar_forecast)
   * [Eco Mode](/docs/webinterface/energy_management/eco_mode)

 * Network
