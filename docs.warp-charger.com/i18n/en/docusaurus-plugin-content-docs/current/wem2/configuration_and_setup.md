# Configuration and Setup

After installation, the WARP Energy Manager 2.0 is configured via its web interface. Guidance
for the first steps (establishing a connection and opening the web interface) can be found
under [First Steps](/docs/tutorials/first_steps).

The following settings are available in the WARP Energy Manager 2.0 firmware:

 * [Status](/docs/webinterface/status)
 * Energy Manager
   * [Display](/docs/webinterface/energy_manager/display)
   * [Energy Analysis](/docs/webinterface/energy_manager/energy_analysis)
   * [Automation](/docs/webinterface/energy_manager/automation)
   * [SD Card](/docs/webinterface/energy_manager/sd-card)
 * Energy Management
   * [Energy Meters](/docs/webinterface/energy_management/energy_meters)
   * [Battery Control](/docs/webinterface/energy_management/battery_control)
   * [Heating](/docs/webinterface/energy_management/heater)
   * [Chargers](/docs/webinterface/energy_management/wallboxes)
   * [Charge Tracker](/docs/webinterface/energy_management/charge_tracker)
   * [PV Excess Charging](/docs/webinterface/energy_management/pv-excess-charging)
   * [Charge Management](/docs/webinterface/energy_management/loadmanagement)
   * [Dynamic Tariff](/docs/webinterface/energy_management/dynamic_tariffs)
   * [Solar Forecast](/docs/webinterface/energy_management/solar_forecast)
   * [Outdoor Temperature](/docs/webinterface/energy_management/temperatures)
   * [Eco Mode](/docs/webinterface/energy_management/eco_mode)
