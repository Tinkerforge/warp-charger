    <td>
        The current state, processed by the charge controller
        
        
        
        - **0** - Not connected
        - **1** - Waiting for charge release
        - **2** - Ready to charge
        - **3** - Charging
        - **4** - Error
    </td>
</tr>

<tr>
    <th scope="row" style={{whiteSpace: "nowrap"}}>`contactor_state`<br/>int</th>
    <td>
        State of the contactor monitoring.
        
        <strong>(Only WARP 1 and WARP 2)</strong> 
        
        - **0** - Not live before and after contactor
        - **1** - Live before, but not live after contactor
        - **2** - Not live before, but live after contactor
        - **3** - Live before and after contactor
        
        <strong>(Only WARP 3)</strong> 
        
        - **00000..11111** - Bit mask indicating the contactor state. From bit 0 to bit 4:
                        - **0** - Contactor L1+N auxiliary contact: (indicates if contactor is switched) 1 = closed; 0 = open
                        - **1** - Contactor L2+L3 auxiliary contact: (indicates if contactor is switched) 1 = closed; 0 = open
                        - **2** - Contactor error: (see contactor_error) 1 = error; 0 = no error
                        - **3** - Contactor coil connection: 1 = One or both contactors should be switched (see phase switching state) 0 = No contactor should be switched
                        - **4** - Phase switching state (indicates which contactors are switched when the coil connection bit is 1). 1 = Both contactors are switched; three-phase. 0 = Only the L1+N contactor is switched; single-phase 
    </td>
</tr>

<tr>
    <th scope="row" style={{whiteSpace: "nowrap"}}>`contactor_error`<br/>int</th>
    <td>
        Error code of the contactor monitoring. A value other than 0 indicates an error.
        
        
        
        - **0** - No error
        
        <strong>(Only WARP 1 and WARP 2)</strong> 
        
        - **1** - Contactor should be closed.<br/>No current before contactor, no current after contactor.<br/>Check power supply.
        - **2** - Contactor should be closed.<br/>Current before contactor, no current after contactor.<br/>Contactor defective?
        - **3** - Contactor should be closed.<br/>No current before contactor, current after contactor.<br/>Check wiring.
        - **4** - Contactor should <strong>not</strong> be closed.<br/>No current before contactor, no current after contactor.<br/>Check power supply.
        - **5** - Contactor should <strong>not</strong> be closed.<br/>No current before contactor, current after contactor.<br/>Check wiring.
        - **6** - Contactor should <strong>not</strong> be closed.<br/>Current before contactor, current after contactor.<br/>Contactor defective?
        
        <strong>(Only WARP 3)</strong> 
        
        - **1** - Contactor L2+L3 not closed.<br/>Both contactors should be closed.
        - **2** - Contactor L1+N not closed.<br/>Both contactors should be closed.
        - **3** - Both contactors not closed.<br/>Both contactors should be closed.
        - **4** - Contactor L2+L3 closed.<br/>Only contactor L1+N should be closed.
        - **5** - Contactor L2+L3 closed, contactor L1+N not closed.<br/>Only contactor L1+N should be closed.
        - **6** - Both contactors closed.<br/>Only contactor L1+N should be closed.
