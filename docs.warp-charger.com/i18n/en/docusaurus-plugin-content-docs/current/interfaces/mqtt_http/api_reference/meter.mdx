# Legacy Electricity Meter API
**Use [`meters` module](/interfaces/mqtt_http/api_reference/meters.mdx) for new developments instead!**
The legacy electricity meter API only supports one electricity meter (the one configured as `linked_meter_slot` in [`meters_legacy_api/config`](/interfaces/mqtt_http/api_reference/meters_legacy_api.mdx#meters_legacy_api_config)) and can only report and process a subset of the supported electricity meter values. The legacy API is emulated by the [`meters_legacy_api` module](/interfaces/mqtt_http/api_reference/meters_legacy_api.mdx) module.
