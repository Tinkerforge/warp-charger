# Credentials

:::note

This option is only available for the WARP Energy Manager, as it does not
have its own user management system.

:::

![image](/img/webinterface/system/wem2-system_login_details.jpeg)

On this page, a username and password can be configured to protect access to
the web interface of the [WARP Energy Manager](/warp_energy_manager/introduction.md).
When login is enabled, access to the web interface and HTTP API is only possible with the correct
login credentials.
