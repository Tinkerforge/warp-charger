# Technical Specification

The WARP2 Charger has the following technical specification:

### Electrical Properties

| Property | Value |
|-|-|
| Charging Standard | DIN EN 61851-1 |
| Charging Power | Adjustable up to 11 kW / 22 kW (depending on variant) |
| Nominal Voltage | 230 V (1-phase) / 400 V (3-phase) AC (depending on variant) |
| Nominal Frequency | 50 Hz |
| Nominal Current | 16 A / 32 A (depending on variant) |
| Power Consumption | Basic / Smart ≤ 3 W; Pro ≤ 5 W (Standby, WLAN active) |
| Operating Temperature | -25 °C to +50 °C (average over 24 h: ≤ 35 °C) |
| Fault Current Detection | DC 6 mA (integrated) |
| Current Measurement | Integrated MID-certified energy meter according to EU Measuring Instruments Directive 2014/32/EU (Pro only) |

### Mechanical Properties

| Property | Value |
|-|-|
| Vehicle Charging Plug | Type 2 |
| Charging Cable Length | 5 m / 7.5 m (depending on variant) |
| Supply Line Cross-Section | 2.5 mm² to 10 mm² |
| Charger Dimensions | 280 × 215 × 95 mm (W/H/D) |
| Protection Class | IP54 (splash-proof, suitable for outdoor use) |

### Software Properties

| Property | Value |
|-|-|
| Access Control | NFC, web interface (depending on variant) |
| Load Management | Max. 32 participants (depending on variant) |
| NFC Tags | 3 included in delivery, max. 32 can be learned (depending on variant) |
| Users | Max. 32 configurable (depending on variant) |
| Interfaces | HTTP, MQTT, Modbus/TCP, OCPP (depending on variant) |
