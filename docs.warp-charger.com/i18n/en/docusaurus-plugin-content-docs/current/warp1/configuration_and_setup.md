# Configuration and Setup

After installation, the WARP1 Charger is configured via its web interface. Help for the
first steps (establishing a connection and opening the web interface) can be found under
[First Steps](/docs/tutorials/first_steps). The Basic variant does not have a web interface.

The following settings are available in the firmware of the WARP1 Charger:

 * [Status](/docs/webinterface/status)
 * Charger
   * [Settings](/docs/webinterface/wallbox/settings)
   * [Charging Status](/docs/webinterface/wallbox/charge_status)
   * [Charge Tracker](/docs/webinterface/wallbox/charge_tracker)
   * [Automation](/docs/webinterface/wallbox/automation)
 * Energy Management
   * [Energy Meters](/docs/webinterface/energy_management/energy_meters)
   * [Battery Control](/docs/webinterface/energy_management/battery_control)
   * [Chargers](/docs/webinterface/energy_management/wallboxes)
   * [PV Excess Charging](/docs/webinterface/energy_management/pv-excess-charging)
   * [Load Management](/docs/webinterface/energy_management/loadmanagement)
 * Network
   * [Settings](/docs/webinterface/network/settings)
   * [WiFi Connection](/docs/webinterface/network/wifi-connection)
   * [WiFi Access Point](/docs/webinterface/network/wifi-accesspoint)
   * [WireGuard](/docs/webinterface/network/wireguard)
 * Interfaces
   * [MQTT](/docs/webinterface/interfaces/mqtt)
   * [Modbus/TCP](/docs/webinterface/interfaces/modbus_tcp)
   * [OCPP](/docs/webinterface/interfaces/ocpp)
 * Users
   * [User Management](/docs/webinterface/users/user_management)
 * System
   * [Settings](/docs/webinterface/system/settings)
   * [TLS Certificates](/docs/webinterface/system/tls-certificates)
   * [Remote Access](/docs/webinterface/system/remote_access)
