# Konfiguration und Einrichtung

WARP Charger und WARP Energy Manager basieren auf derselben Software. Folgende Einstellungen sind für den WARP Energy Manager verfügbar:
 
 * Status
 * EnergyManager
   * Display
   * Energiebilanz
   * Automatisierung
   * SD-Karte
 * Energiemanagement
   * Stromzähler
   * Wallboxen
   * Heizung
   * PV-Überschussladen
   * Lastmanagement
   * Dynamischer Strompreis
   * PV-Ertragsprognose
   * Eco-Modus
 * Netzwerk
   * Einstellungen
   * WLAN-Verbindung
   * WLAN-Access-Point
   * LAN-Verbindung
   * WireGuard
 * Schnittstellen
   * MQTT
 * System
   * Einstellungen
   * TLS-Zertifikate
   * Fernzugriff
   * Systemzeit
   * Zugangsdaten
   * Ereignis-Log
   * Firmware-Aktualisierung
