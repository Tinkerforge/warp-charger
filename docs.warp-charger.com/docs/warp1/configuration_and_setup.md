# Konfiguration und Einrichtung

Nach der Installation wird der WARP1 Charger über sein Webinterface konfiguriert.
Eine Hilfestellung für die ersten Schritte (Verbindung herstellen und das Webinterface
öffnen) findet sich unter [Erste Schritte](/docs/tutorials/first_steps). Die Variante Basic
besitzt kein Webinterface.
