# Konfiguration und Einrichtung

WARP Charger und WARP Energy Manager basieren auf derselben Software. Folgende Einstellungen sind für den WARP Charger verfügbar:
 
 * Status
 * Wallbox
   * Einstellungen
   * Ladestatus
   * Ladetracker
   * Automatisierung
 * Energiemanagement
   * Stromzähler
   * Wallboxen
   * PV-Überschussladen
   * Lastmanagement
   * Dynamischer Strompreis
   * PV-Ertragsprognose
   * Eco-Modus
 * Netzwerk
   * Einstellungen
   * WLAN-Verbindung
   * WLAN-Access-Point
   * LAN-Verbindung
   * WireGuard
 * Schnittstellen
   * MQTT
   * Modbus/TCP
   * OCPP
 * Benutzer
   * NFC-Tags
   * Benutzerverwaltung
 * System
   * Einstellungen
   * TLS-Zertifikate
   * Fernzugriff
   * Systemzeit
   * Ereignis-Log
   * Firmware-Aktualisierung
