# Technische Spezifikation

Der WARP Energy Manager 1.0 besitzt folgende technische Spezifikation:

### Elektrische Eigenschaften

| Eigenschaft | Wert |
|-|-|
| Nennspannung | 230 V AC |
| Nennfrequenz | 50 Hz |
| Eigenverbrauch min. | 1,1 W (LAN aktiv, WLAN-Fallback, Relais aus, LED aus) |
| Eigenverbrauch max. | ca. 2 W (LAN aktiv, WLAN ein, Relais ein, LED ein) |
| Schutzklasse | II |

### Mechanische Eigenschaften

| Eigenschaft | Wert |
|-|-|
| Abmessungen | 70 × 90 × 63 mm (B/H/T) |
| Montageort | Schaltschrank |
| Montageart | Tragschiene |
| Betriebstemperatur | 0 °C bis +30 °C |

### Funktionen

| Eigenschaft | Wert |
|-|-|
| PV-Überschussladen | max. 10 WARP Charger |
| Lastmanagement | max. 10 WARP Charger |
| Netzwerk | LAN, WLAN |
| Schnittstellen | HTTP, MQTT |

WARP Charger und WARP Charger 2 werden jeweils in den Varianten Smart und Pro
unterstützt.
