# LAN-Verbindung
**Benötigt das Feature <a href="#features_ethernet"><code>"ethernet"</code></a>.**
Mit dem `ethernet`-Modul und passender Hardware kann eine Verbindung zu einem LAN hergestellt werden.

## `ethernet/config` {#ethernet_config_warp2}


Die Konfiguration der LAN/Ethernet-Verbindung..
:::info[Beispiel]

```bash
# $HOST z.B. warp2-AbCd
```

#### Lesen
```bash
curl http://$HOST/ethernet/config
```
```jsx
{
  "enable_ethernet": true,
  "ip": "0.0.0.0",
  "gateway": "0.0.0.0",
  "subnet": "0.0.0.0",
  "dns": "0.0.0.0",
  "dns2": "0.0.0.0"
}
```



#### Schreiben
```bash
curl http://$HOST/ethernet/config -d '{
  "enable_ethernet": true,
  "ip": "0.0.0.0",
  "gateway": "0.0.0.0",
  "subnet": "0.0.0.0",
  "dns": "0.0.0.0",
  "dns2": "0.0.0.0"
}'
```



        </TabItem>

mosquitto_sub -v -C 1 -h $BROKER -t $PREFIX/ethernet/config
