## `ntp/config` {#ntp_config_any}


Die Konfiguration der Netzwerkzeitsynchronisierung.

    <Tabs groupId="apiType" queryString className="hidden-tabs">

```bash
# $HOST z.B. warp-AbCd
```

```bash
curl http://$HOST/ntp/config
```
```jsx
{
  "enable": true,
  "use_dhcp": true,
  "timezone": "Europe/Berlin",
  "server": "time.cloudflare.com",
  "server2": "ptbtime3.ptb.de"
}
```

```bash
curl http://$HOST/ntp/config -d '{
  "enable": true,
  "use_dhcp": true,
  "timezone": "Europe/Berlin",
  "server": "time.cloudflare.com",
  "server2": "ptbtime3.ptb.de"
}'
```
